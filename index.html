Rules:-
This game is online i.e you can play anywhere, anytime.
Each level contains clues in many forms. You need to find the clues, follow them, find the answer and type it down to reach next level.
Clues can be anywhere in the page. It can be in a comment inside source code, title of the page or in the image or page URL. You don't need any programming skills (at least no web designing skills) to find them or crack the level.
Once you find the answer you might have to enter it in URL itself or you might have to type it in space provided or some events in your page can take you to next level.
If you see the same level again that could mean your answer is wrong. There's nothing wrong with website.
You might have to use tools and softwares both online and offline in order to make it to next level.
